# EmailSender
In het eerste scherm wordt de verbinding met de SMTP Server van je email provider ingesteld.
<br>
Met knop "Testen" kan verbinding worden getest, deze verbinding wordt gebruikt voor verzending van de mail.

### Tabs
- In tab "Ontvangers" worden de email adressen opgegeven.
- In tab "Bericht" wordt berichttekst opgegeven.
- In tab "Log" worden de statusmeldingen getoond.
- In tab "EML Opslag" worden de berichbestanden (in eml formaat) getoond.
- In tab "Bijlagen" kunnen bijlagen worden toegevoegd.
#### Bijlagen
- Algemene bijlagen worden naar alle ontvangers verstuurd.
- Persoonlijke bijlagen alleen naar de betreffende ontvanger.

### Knoppen
- Met de knop "Verzenden" worden de mails gegenereerd en verzonden.
- Met de knop "Opslaan EML" worden de mails gegenereerd en in EML-formaat opgeslagen (zie tab "EML Opslag").
- Met de knop "Testen" kan de verbinding met de ingestelde SMTP server worden getest.

Als het programma via OSMMapper wordt gestart, dan worden de "Ontvangers" en de "Persoonlijke bijlagen" gevuld met Marker informatie en foto's.
